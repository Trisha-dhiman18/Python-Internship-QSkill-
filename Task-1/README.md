# Task 1: Student Performance Analysis

## Objective
Analyze student performance data using Python to extract meaningful insights.

## Dataset
The dataset contains demographic details and subject-wise scores of 1000 students.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data loading and inspection
- Average score calculation
- Gender-wise and preparation-based analysis
- Visualizations using bar charts, scatter plots, heatmaps, and pie charts

## Key Insights
- Test preparation improves overall performance
- Reading and writing scores are strongly correlated
