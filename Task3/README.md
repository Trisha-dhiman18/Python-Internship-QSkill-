# Matrix Operations Tool

## Description
This project is developed as **Internship Task 3**.  
It is a **Matrix Operations Tool** created using **Python and NumPy**.

The project is implemented in two ways:
- Using **Tkinter (GUI version)**
- Without Tkinter (**Command Line version**)

---

## Operations Performed
- Matrix Addition  
- Matrix Subtraction  
- Matrix Multiplication  
- Matrix Transpose  
- Determinant Calculation  

---

## Technologies Used
- Python  
- NumPy  
- Tkinter (only for GUI version)
